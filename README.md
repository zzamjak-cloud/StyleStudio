# StyleStudio - AI 이미지 생성 워크스테이션

AI를 활용한 게임 아트 제작을 위한 데스크톱 애플리케이션

## 버전 0.3.2 업데이트 (2024-04-15)

### 새로운 기능

#### 1. 기본 세션 개선
- **채팅 입력창 개선**: 기본 3줄 높이로 시작하며 내용에 따라 자동 확장 (최대 10줄)
- **레이아웃 분할**: 좌측 채팅 영역과 우측 AI 설정 패널로 분리
- **AI 설정 패널**: 모델 선택, 이미지 비율, 크기, 그리드 설정을 우측 패널에서 관리

#### 2. 컨셉 세션 (신규)
모바일 게임 컨셉 아트 생성을 위한 전문 세션 타입

**좌측 패널 - 게임 정보 입력**
- 참조 이미지 업로드 (드래그 앤 드롭 지원)
- 게임 장르 선택 (프리셋 + 커스텀 입력)
- 게임 플레이 스타일 입력
- 레퍼런스 게임 목록
- 아트 스타일 선택

**우측 패널 - 이미지 생성**
- 프롬프트 입력 (선택사항, 비워두면 자동 생성)
- 모델 선택 (나노바나나 프로/나노바나나 2)
- 생성 설정 (비율, 크기, 그리드 레이아웃)

**하단 히스토리 패널**
- 드래그로 높이 조절 가능
- 생성된 이미지 썸네일 보기
- 이미지 저장 및 삭제 기능

## 기술 스택

- **프론트엔드**: React + TypeScript + Tailwind CSS
- **데스크톱**: Tauri v2
- **상태 관리**: Zustand
- **AI API**: Google Gemini API

## Recommended IDE Setup

- [VS Code](https://code.visualstudio.com/) + [Tauri](https://marketplace.visualstudio.com/items?itemName=tauri-apps.tauri-vscode) + [rust-analyzer](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer)

## 빌드 가이드

### 로컬 빌드
1. `tauri.conf.json`의 `createUpdaterArtifacts`를 `false`로 변경
2. `npm run tauri build` 실행

### 릴리스 빌드 (태그 푸시)
1. `tauri.conf.json`의 `createUpdaterArtifacts`를 `true`로 변경
2. 커밋 및 태그 푸시
3. GitHub Actions가 자동으로 빌드 및 릴리스 생성
